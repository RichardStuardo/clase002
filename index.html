1. Crear el repositorio en github
  2. Crear la carpeta donde se almacenará nuestro proyecto
  3. Abrir la CARPETA(no un archivo) con visual studio code
  4. Abrir terminal integrada
  5. git init
  6. git remote add origin https://github.com/nombredeusuario/repositorioEsta instrucción agrega el origen
  7. git add nombredearchivoesta instrucción agrega nuestros cambios
  8. git commit -m origin masteragrega un nombre a nuestra piscina de cambios
  9. git push -u origin mastercrea la rama de master en el repositorio y sube los cambios
  10. finalmente, cuando seguimos haciendo cambios, ejecutamos : git push(luego de haber agregado los cambios y realizado commit)
